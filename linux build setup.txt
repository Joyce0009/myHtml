LinuxOS(16.04) build setup
=========================
Network - Setting
	Ethernet>MTU: automatic
	802.1x security
		Authentication: Protected EAP(PEAP)
		"checked" No Ca certificate is required
		PEAP version: Automatic
		Inner authentication: MSCHAPv2
		Username, Password


1. linxu沒有網路
	$ nmtui-connect 
		(跳出來網路介面，進行網路啟用)

2. ssh -p 29418 joyce_lin@amax01.corpnet.asus
   Unable to negotiate with legacyhost: no matching key exchange method found.
	Their offer: diffie-hellman-group1-sha1
	
	參考https://www.openssh.com/legacy.html，在~/.ssh/config
	來enable 設定 "diffie-hellman-group1-sha1"
	
	把 下列這些加到 ~/.ssh/config

	Host amax01.corpnet.asus
	KexAlgorithms +diffie-hellman-group1-sha1

3. 把adb路徑加到PATH
   在 ~/.bash_profile加
   export PATH=$PATH:/home/joyce-lin/Android/Sdk/platform-tools
   
   $ source ~/.bash_profile

4. Set up a device for development (USB access)
	(1) Log in as root and create this file: /etc/udev/rules.d/51-android.rules.
	(2-1) Using $lsusb 檢查，ASUS Device是0b05
	(2) add each ASUS vendor to the file
	    SUBSYSTEM=="usb", ATTR{idVendor}=="0b05", MODE="0600", OWNER="joyce-lin"
	// google developer 上寫 MODE="0666" 是read/write
	(3) $ chmod a+r /etc/udev/rules.d/51-android.rules
